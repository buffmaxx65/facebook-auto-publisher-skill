# Facebook Auto Publisher — OpenClaw Skill

Skill untuk mempublikasikan, menjadwalkan, dan melaporkan posting Facebook untuk workflow affiliate.

## Fungsi utama

- Menyiapkan rencana posting
- Memposting post utama dan komentar lanjutan
- Menggunakan Facebook API, scheduler resmi, browser automation, atau output manual
- Mengambil link hasil posting
- Menjaga ritme posting agar tidak spam

## Install

```bash
git clone https://github.com/buffmaxx65/facebook-auto-publisher-skill \
  ~/.openclaw/skills/facebook-auto-publisher
```

## Cara pakai

```bash
openclaw agent --message "Jadwalkan dan posting konten Facebook ini ke Page saya jam 20.00"
```

## Catatan

Skill ini tidak boleh posting tanpa izin eksplisit user dan tidak boleh bypass proteksi login Facebook.
